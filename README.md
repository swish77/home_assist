# 🏠 HomeAssist

HomeAssist is a Flutter app template for a home services marketplace. It allows users to browse home services, receive AI-powered service recommendations, book services, and view their bookings through a clean and intuitive interface.

The project was built to practice scalable Flutter development with a modular architecture, reusable UI components, and REST API integration.

---

## ✨ Features

* Browse services by category
* View popular services
* AI-powered service recommendations
* Book home services
* View upcoming and completed bookings
* REST API integration for dynamic data
* Responsive Material 3 UI

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/home_assist.git
```

2. Install dependencies.

```bash
flutter pub get
```

3. Run the project.

```bash
flutter run
```
